# rn-idle-timer

An API for disabling the screen idle timer for iOS and Android

## Installation

```sh
npm install rn-idle-timer
```

## Usage

```js
import { multiply } from 'rn-idle-timer';

// ...

const result = await multiply(3, 7);
```
